# MarketX — Telegram Mini App

Маркетплейс для Telegram.

## Деплой на GitHub Pages

1. Залей все файлы в репозиторий
2. Settings → Pages → Branch: `main` → Folder: `/ (root)`
3. Сохрани — через ~2 минуты сайт будет доступен
4. URL вида: `https://YOUR_USERNAME.github.io/REPO_NAME/`

## Файлы

- `index.html` — Mini App
- `_config.yml` — конфиг GitHub Pages
- `.nojekyll` — отключает Jekyll (обязательно!)
- `404.html` — редирект для роутинга
